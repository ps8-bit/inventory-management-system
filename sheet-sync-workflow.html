พอส่งสินค้าในแอป → ติ๊ก "✅ ส่งแล้ว" กลับเข้าชีตให้อัตโนมัติ

⚙️ Apps Script (ตั้งเวลา)

➜
4
📗 ชีตติ๊กให้
เติมคอลัมน์ ✅ ส่งแล้ว · เลขพัสดุ · ขนส่ง · วันที่
🔑 กุญแจสำคัญ 2 อย่าง
• คีย์จับคู่แถว: ใช้ SO-id (คอลัมน์แรกของชีต) เป็นตัวเชื่อมว่าแถวไหน = ออเดอร์ไหน
• ความปลอดภัย: ใส่ service_role key ไว้ใน Apps Script (ฝั่ง Google) เท่านั้น — ห้ามใส่ในโค้ดเว็บ/ไม่ commit ลง git
